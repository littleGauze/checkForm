<input type="text" name="username" req="用户名不能为空!" > :(req="用户名不能为空!")字段必须 且 提示信息会替换为req属性的值，即[用户名不能为空!]
<input type="text" name="userpass" req passgt6> :(req)字段必须 且 (passgt6)验证密码必须大于6位字符
<input type="text" name="confirmpass" req cpass=userpass> :(req)字段必须 且 (cpass=userpass)验证两次密码输入是否一致[userpass:为首次输入密码表单的name]
<input type="text" name="contact" req tel telOnly> :(req)字段必须 且 (tel)验证座机或者手机号码 (telOnly)只允许输入数字和短横线[-]
<input type="text" name="email" req email> :(req)字段必须 且 (email)验证邮箱地址
<input type="text" name="salary" numOnly> :(numOnly)只允许输入数字

除了 [ *Only, cpass ] 其他的表单验证都可以设置相应属性的值，该值会当做提示信息显示。如上面的 用户名 字段
